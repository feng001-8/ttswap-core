# toUint128
Safely converts a uint256 to a uint128


```solidity
function toUint128(uint256 a) pure returns (uint128 b);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`a`|`uint256`|The uint256 value to convert|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`b`|`uint128`|converted uint128 value, or 0 if overflow|
